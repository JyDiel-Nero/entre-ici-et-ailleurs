# 🎵 Tutoriel Audio — Entre ici et ailleurs

## Ajouter un enregistrement audio

1. Allez sur `/admin` et connectez-vous
2. Cliquez **🎵 Audio** → **Gérer les enregistrements**
3. Cliquez **Add tracks**
4. Remplissez les champs :

| Champ | Description |
|-------|-------------|
| **Titre** | Nom de l'enregistrement |
| **Auteur** | Par défaut « J.Y.D. » — modifiable |
| **Description** | Courte description (optionnel) |
| **Fichier MP3** | Uploadez un MP3 OU laissez vide si vous utilisez un lien |
| **Lien YouTube** | URL YouTube complète — sera intégré comme vidéo |
| **Lien Instagram** | URL d'un post Instagram (lien externe) |
| **Lien TikTok** | URL d'un TikTok (lien externe) |
| **Autre lien** | N'importe quel autre lien (SoundCloud, Spotify, etc.) |
| **Texte du lien** | Texte affiché pour « Autre lien » (ex: « Écouter sur Spotify ») |
| **Image de couverture** | Image affichée au-dessus du lecteur (optionnel) |
| **Ordre** | Numéro pour le tri (1 = premier) |
| **Publié** | Cocher pour rendre visible |

5. Cliquez **Save**

## Audio vs YouTube vs liens

Vous pouvez combiner les trois sur un même enregistrement :
- **MP3** = lecteur audio intégré directement sur le site
- **YouTube** = vidéo intégrée (iframe) sous le lecteur audio
- **Instagram/TikTok** = liens cliquables qui ouvrent dans un nouvel onglet

Vous pouvez aussi utiliser un seul des trois. Par exemple, si vous n'avez pas de MP3 mais une vidéo YouTube, laissez le champ MP3 vide et remplissez uniquement YouTube.

## Cacher la section Audio

Si vous n'avez pas encore de contenu audio :
1. ⚙️ Paramètres → décochez **Afficher Audio**
2. La section et le lien dans la navigation disparaissent
3. Quand vous êtes prêt, recochez-la
