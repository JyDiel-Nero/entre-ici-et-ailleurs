# 📚 Tutoriel Œuvres — EIA

## Ajouter une œuvre
1. CMS → 📚 Œuvres → Gérer les œuvres
2. Add oeuvres
3. Choisissez le type : 📖 Recueil, 🎤 Album, ou 🎨 Peinture
4. Remplissez : titre, description, image, prix
5. Ajoutez les liens d'achat (tous optionnels) :
   - Amazon, Bookelis, Fnac, Spotify, ou lien personnalisé
6. Ordre : numéro pour le tri (1 = premier)
7. Publié : cocher pour rendre visible
8. Save

## Activer la section Œuvres
La section est **cachée par défaut**. Pour l'activer :
1. CMS → 📂 Sections
2. Cochez ✅ Afficher Œuvres
3. Save

## Filtres
Sur le site, les visiteurs peuvent filtrer par type : TOUS, 📖 Recueils, 🎤 Albums, 🎨 Peintures.

## Renommer la section
CMS → 📂 Sections → champ "Nom : Œuvres" → changez en ce que vous voulez.
