# ➕ Tutoriel Sections personnalisées — EIA

## Créer une section
1. CMS → ➕ Sections perso → Gérer les sections
2. Add sections
3. Remplissez :
   - **Identifiant** : minuscules + tirets (ex: reflexions, citations, projets)
   - **Nom** : affiché dans la navigation
   - **Type** : Blog, Galerie, Page texte, ou Liens
   - **Visible** : décochez pour cacher jusqu'au lancement
4. Si type Blog : ajoutez des sous-catégories
5. Ajoutez du contenu dans les éléments
6. Save

## Types de sections

### 📝 Blog (articles avec catégories)
- Chaque élément a : titre, texte, image, lien, catégorie
- Les sous-catégories permettent de filtrer

### 🖼️ Galerie (grille d'images)
- Chaque élément a : titre, image
- Affichage en grille masonry

### 📄 Page texte (contenu libre)
- Un seul champ de contenu en Markdown
- Supporte **gras**, *italique*, et [liens](url)

### 🔗 Liens (liste de liens)
- Chaque élément a : titre, texte, lien
- Affiché comme une liste cliquable stylisée

## La section apparaît automatiquement
Quand Visible est coché, la section s'ajoute à la navigation et au footer automatiquement. Pas besoin de toucher au code.
